# titanicdataanalysis
**Overview**
This project analyzes the Titanic dataset to understand the factors that influenced passenger survival.
It includes data cleaning, exploratory data analysis (EDA), and visualization to extract meaningful insights.

**Objectives**
1.Clean and preprocess raw data
2.Handle missing values and inconsistencies
3.Perform exploratory data analysis (EDA)
4.Visualize patterns and relationships
5.Identify key factors affecting survival

**Tools and Technologies**
Python
Pandas
NumPy
Matplotlib
Seaborn

**Dataset**
Source: Kaggle Titanic Dataset
File used: train.csv
kaggle link: https://www.kaggle.com/c/titanic/data


**Key Insights**
Females had a significantly higher survival rate than males
Passengers in 1st class had better survival chances
Younger passengers had slightly higher survival probability


**Conclusion**]
The analysis shows that gender, passenger class, age, and fare were important factors influencing survival on the Titanic.
